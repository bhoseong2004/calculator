새롭게 추가된 특수 기능 안내

  • n! 팩토리얼 (Factorial)
    현재 입력된 정수의 팩토리얼 값을 계산합니다. (170 이하의 정수만 가능)
  • 1/x 역수 (Reciprocal)
    현재 입력된 숫자를 1로 나눈 값을 구합니다. (0 입력 시 에러)
  • log 상용로그 (Logarithm)
    밑이 10인 로그 값을 계산합니다. (양수만 가능)
  • RND 난수 생성 (Random Number)
    0과 1 사이의 무작위 소수점 난수를 생성합니다.
